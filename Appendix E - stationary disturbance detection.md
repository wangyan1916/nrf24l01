# 稳态扰动检测

在增强型 ShockBurst™ 中，建议仅当 PTX 设备未成功让数据包通过时才使用载波检测功能，如单个数据包的 `MAX_RT` IRQ 所示，以及多个数据包丢失时的数据包丢失计数器 (`PLOS_CNT`) 所示。

如果 PTX 设备中的 `PLOS_CNT` 指示丢包率较高，则可以将设备配置为 PRX 设备一段时间（Tstbt2a + CD 过滤器延迟 = 130μs+128μs = 258μs）以检查 CD。

如果 CD 较高（堵塞情况），则应更改频道。 如果 CD 较低（超出范围或被 WLAN 等宽带信号干扰），它可能会继续在同一频道上运行，但您必须执行其他调整（对`RF_CH`的虚拟写入会清除`PLOS_CNT`）
